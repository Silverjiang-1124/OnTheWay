# agent/ — 行程规划对话记录

此目录用于存储与我（Claude）关于具体行程规划的对话记录。

## 使用方式

在项目根目录打开 Claude，直接说：

> 帮我制定一个武功山重装徒步行程

我会在这里创建对应的行程规划文件，包含完整的行程方案、装备建议、路线分析等。

## 目录结构

```
agent/
├── README.md          # 本文件
├── trips/             # 具体行程规划
│   ├── wugongshan.md  # 示例: 武功山行程
│   └── ...
└── templates/         # 规划模板 (可选)
```

## 建议话题

- 制定全新行程
- 已有行程的装备优化建议
- 路线难度分析
- 天气/季节风险评估
- 打包策略建议
